Lead Developer | C++ Project | Spring 2024

Created a C++ simulation that helped a miner escape a grid-based mine by choosing efficient paths, comparing heap implementations, and handling TNT detonation logic.

  • Used priority queues to optimize for lower rubble values and faster escape routes.
  • Implemented binary heap, Fibonacci heap, and pairing heap approaches to compare insertion and retrieval tradeoffs.
  • Added custom comparator logic to support complex tie-breaking and improve pathfinding accuracy.
